# MyCityGov Secure AWS Platform

A portfolio project focused on designing and implementing a secure multi-tier AWS environment for hosting the MyCityGov application.

The MyCityGov application included in this repository is based on a previous academic project.

The focus of this repository is on the secure cloud infrastructure and deployment environment, not on redesigning the application itself.

## Project Goals

- Secure multi-tier AWS architecture
- Network segmentation
- IAM least privilege
- Secrets management
- DevSecOps pipeline
- Logging, monitoring, and threat detection
- Basic incident response automation

## Current Status

Early design and architecture phase.
